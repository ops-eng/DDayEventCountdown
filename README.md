# DDay Event Countdown
## Your Events Countdown to post in Social Media

### DDay Event Countdown helps you create aesthetic countdown cards for the moments that matter.

Create countdowns for trips, weddings, birthdays, launches, holidays, sales, school events, and personal milestones. Add a title, date, emoji, theme, and background, then preview your countdown card in social-ready formats.

### Includes:

- 10 pastel themes
- 37 background options
- 36 quick emojis, plus custom emoji keyboard input
- Multiple social export presets
- PNG sharing and Photos saving
- Light, Dark, and System appearance modes
